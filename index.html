Finance Backpack

Finance Backpack is your very own app for financialcalculators, tutorials and useful links.
If you are from a non-commerce background, it will help you in understanding the basic financial system. You can further read on topics which you might need to know more about.

Along with tutorials, Financial Backpack provides you with various Finance calculators which you might need on regular bases. The basic interest rate calculation does not now require you to memorize formulas; with the help of our app you can calculate them at ease.
 

The available calculators are:
1. GST calculator
2. EMI calculator
3. Loan comparator
4. Fixed deposit calculator
5. Present value calculator
6. Future value calculator

We plan to add new calculators as per user’s need.
